# CodeAlpha_URL_Shortener
[cite_start]This is Task 1: Simple URL Shortener for the CodeAlpha Backend Development Internship[cite: 24].

## [cite_start]Features [cite: 27, 29]
- API to generate unique short codes from long URLs.
- Database storage for URL mapping.
- Automatic redirection to the original long URL.
- [cite_start]Basic frontend for user interaction[cite: 30].

## [cite_start]Tech Stack [cite: 5, 25]
- **Backend:** Node.js & Express.js
- **Database:** MongoDB
- **View Engine:** EJS
